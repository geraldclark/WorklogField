Overview

A custom POST endpoint used to fetch user display names for worklog fields.

Input Parameters

Name Type Description
ids Array List of user ids to fetch display names for.

Result

Name Type Description
<user id> String The users full name.

Output Example

{
    "seed_chris_id":"Chris Olliver",
    "seed_jim_id":"Jim Brennan"
}
